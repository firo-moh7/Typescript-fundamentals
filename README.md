# TypeScript Fundamentals

This repository contains my Day 1 TypeScript practice as part of my full-stack development roadmap.

## Topics Covered

* Basic Types

  * string
  * number
  * boolean

* Interfaces

* Optional Properties

* Typed Functions

* Arrays of Objects

* forEach Iteration

## Exercises

### Study Group Example

Created a `GroupMatcher` interface and used typed objects to represent study groups.

### User Management Example

Created a `User` interface and stored multiple users in a typed array.

Implemented a function to display user information using TypeScript type annotations.

## Technologies

* TypeScript
* Node.js
* ts-node

## How to Run

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npx ts-node day1.ts
```

## Learning Outcome

After completing this project, I can:

* Define and use interfaces
* Create typed arrays
* Write typed functions
* Work with arrays of objects
* Understand the difference between a single object type (`User`) and an array of objects (`User[]`)
